# 欢迎使用 VNote
一个舒适的笔记平台。

更多信息，请访问 [**VNote 主页**](https://vnotex.github.io/vnote)。

## VNote 4
VNote 4 是 VNote 3 的全新重构版本，带来了精致的界面和更加强大的功能，包括：

* 基于 git 的笔记本同步后端
* 支持从远程 URL 打开笔记本
* 全新的原始笔记本（Raw Notebook），可自由浏览磁盘上的任意文档

只需点击这里的按钮，即可创建笔记本、打开笔记本，或转换并导入 VNote 3 的笔记本。尽情享受吧！
